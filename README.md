# Anisotropic-Polarization-Filtering-
In this experiment, an LCD privacy protection filter (directional micro-louver structure) was used as the input, and a combination of a 0° square polarizer and a 45° rotated diamond-shaped (rotated square) polarizer was placed at the output stage. White backlight was passed through the system.
# Quasi-Discrete Optical Modes in Privacy Filter + Rotated Polarizer System

**Experimental observation of anisotropic polarization filtering creating bright central clusters and regular discrete spot arrays (satellite-like patterns).**

When a privacy protection film (micro-louver) is combined with a square polarizer and a 45° rotated diamond polarizer, unexpected **quasi-discrete angular modes** emerge — far beyond simple moiré fringes or speckle.

### Key Phenomenon
- Square + Square → Only fringe suppression  
- Square + 45° Diamond → Bright central cluster + surrounding discrete pink spot array with partial lattice ordering

This appears to be a result of **boundary geometry mismatch**, **anisotropic spatial filtering**, and **near-field mode coupling**.
